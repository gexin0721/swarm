# DroneSwarm 部署指南

本文档提供详细的环境配置和部署步骤。

---

## 一、环境要求

### 1.1 硬件要求

| 设备 | 型号/规格 | 用途 |
|------|-----------|------|
| 主控板 | ESP32-S3 | 无人机飞控 |
| 姿态传感器 | JY60 | 三轴姿态数据 |
| GPS模块 | NEO-M8N | 室外定位 |
| 显示屏 | 0.96" OLED | 状态显示 |
| 电机 | 8520空心杯×4 | 四轴推进 |
| 麦克风 | USB/ALSA兼容 | 语音输入 |
| 上位机 | x86_64/ARM64 | ROS运行 |

### 1.2 软件依赖

**无人机端：**
- ESP-IDF v5.0+
- Python 3.8+（用于idf.py）

**上位机端：**
- Ubuntu 20.04 LTS
- ROS Noetic
- Python 3.8+
- GCC 9.0+

## 二、ESP32固件部署

### 2.1 安装ESP-IDF

```bash
# 克隆ESP-IDF
mkdir -p ~/esp
cd ~/esp
git clone -b v5.0 --recursive https://github.com/espressif/esp-idf.git

# 安装工具链
cd esp-idf
./install.sh esp32s3

# 设置环境变量
. ./export.sh
```

### 2.2 编译固件

```bash
cd Software/UAV_ESP

# 设置目标芯片
idf.py set-target esp32s3

# 编译
idf.py build
```

### 2.3 烧录固件

```bash
# 烧录并监控串口输出
idf.py -p /dev/ttyUSB0 flash monitor

# Windows下使用COM端口
idf.py -p COM3 flash monitor
```

### 2.4 串口调试

- 波特率：115200
- 数据位：8
- 停止位：1
- 校验：无

## 三、ROS上位机部署

### 3.1 安装ROS Noetic

```bash
# 设置sources.list
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

# 设置密钥
sudo apt install curl
curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -

# 安装ROS
sudo apt update
sudo apt install ros-noetic-desktop-full

# 环境配置
echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc
source ~/.bashrc
```

### 3.2 一键环境配置

**重要：首次使用必须运行此脚本**

```bash
cd Software/Hive_control_ROS/环境
sudo chmod +x robot.sh
sudo ./robot.sh
```

**脚本自动完成：**
- ROS依赖包安装（amcl、gmapping、mavros等）
- Python依赖安装（websocket-client、vosk、sounddevice、paho-mqtt）
- Vosk语音模型下载
- cJSON库编译安装
- 串口设备权限配置

### 3.3 安装MQTT Broker

```bash
# 安装Mosquitto
sudo apt install mosquitto mosquitto-clients

# 启动服务
sudo systemctl enable mosquitto
sudo systemctl start mosquitto

# 测试连接
mosquitto_sub -t "test" &
mosquitto_pub -t "test" -m "hello"
```

### 3.4 编译ROS功能包

```bash
cd Software/Hive_control_ROS

# 编译
catkin_make

# 设置环境
source devel/setup.bash
```

### 3.5 启动顺序

```bash
# 1. 确保MQTT Broker运行
sudo systemctl status mosquitto

# 2. 启动ROS核心
roscore

# 3. 启动MQTT-ROS桥接（新终端）
rosrun mqtt_ros_bridge mqtt_ros_bridge_node

# 4. 启动语音系统（新终端）
roslaunch voice voice.launch
```

## 四、AI助手配置

### 4.1 API密钥配置

**方式一：Web界面配置（推荐）**

```bash
cd Software/Hive_control_ROS/src/AI/scripts/window
python app.py
```

浏览器访问 `http://localhost:5000` 进行配置。

**方式二：手动编辑配置文件**

编辑 `src/AI/scripts/module/AICore/role/secret_key.json`：

```json
{
  "deepseek": {
    "api_key": "your_api_key",
    "base_url": "https://api.deepseek.com",
    "model": "deepseek-chat"
  },
  "qwen": {
    "api_key": "your_api_key",
    "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "model": "qwen-turbo"
  }
}
```

### 4.2 运行AI助手

```bash
cd Software/Hive_control_ROS/src/AI/scripts
python ai_assistant.py
```

## 六、常见问题排查

### 6.1 编译错误

**问题：ESP-IDF找不到组件**
```bash
# 解决：更新子模块
cd ~/esp/esp-idf
git submodule update --init --recursive
```

**问题：catkin_make失败**
```bash
# 解决：安装缺失依赖
rosdep install --from-paths src --ignore-src -r -y
```

### 6.2 通信问题

**问题：MQTT无法连接**
```bash
# 检查Broker状态
sudo systemctl status mosquitto

# 检查端口
netstat -tlnp | grep 1883

# 测试连接
mosquitto_pub -t "test" -m "hello"
```

**问题：串口权限不足**
```bash
sudo usermod -a -G dialout $USER
# 重新登录生效
```

### 6.3 语音问题

**问题：麦克风无法识别**
```bash
# 检查ALSA配置
cat /proc/asound/cards
# 安装pulseaudio
sudo apt install pulseaudio pavucontrol
```
